# CM3D2.AddModsSlider.Plugin

フリーコメント欄の各modパラメータをスライドバーで操作する UnityInjector 用プラグイン

## 説明

その２>>146以降で発表されている「[CM3D2.AddModsSlider.Plugin](https://github.com/CM3D2-01/CM3D2.AddModsSlider.Plugin)」をダウンロード、コンパイルします

以下のコマンドで、ダウンロードおよびコンパイルを行います

```
download.bat
compile.bat
```

## なんでこのプラグインだけ？

MaidVoicePitch、ExternalSaveDataと関係が深く、また主に当リポジトリ管理者の動作テストが楽になるため、入れてあります。
